       - label: str = "" (for logging/debugging)

    2. WriteQueue class:
       - __init__(maxsize: int = 1000): creates asyncio.Queue[WriteJob | None] with maxsize, _consumer_task = None, _running = False
       - async start(): if not running, set _running=True, create_task(_consume())
       - async stop(): if running, set _running=False, put None sentinel, await consumer task
       - async submit(coro_factory, label=""): create Future via get_running_loop().create_future(), put WriteJob on queue, return await future
       - async _consume(): loop pulling from queue, break on None sentinel, try/except around coro_factory() execution, set_result/set_exception on future, task_done() in finally
       - @property pending -> int: return self._queue.qsize() for monitoring

    Use structlog for logging (consistent with project convention).

       - __init__(model_name="text-embedding-3-small", *, api_key: str | None = None, dimension: int | None = None)
       - Lazy client initialization: _client is None until first embed() call. Use _ensure_client() pattern matching FastEmbedProvider.
       - Properties: model_name, model_version (return f"openai-{self._model_name}"), dimension
