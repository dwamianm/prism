  <name>Task 1: Build context packing module</name>
  <files>src/prme/retrieval/packing.py</files>
  <action>
    Create `src/prme/retrieval/packing.py` implementing Stage 6 per RFC-0006 §2-5.

    **Function: `def estimate_token_cost(text: str, chars_per_token: float = 4.2) -> int`**
    Character-based estimation (RFC-0006 Method 2): `math.ceil(len(text) / chars_per_token)`.
    This is the MVP approach — tiktoken integration can be added later.

    **Function: `def compute_str(candidate: RetrievalCandidate) -> float`**
    Signal-to-Token Ratio per RFC-0006 §2: `composite_score / max(token_cost, 1)`.
    Higher STR = more information per token = better value.

    **Function: `def select_representation(candidate: RetrievalCandidate, available_tokens: int, min_fidelity: RepresentationLevel, chars_per_token: float = 4.2) -> tuple[RepresentationLevel, int]`**
    Determine the best representation level that fits the available budget:
    - FULL: full content text
    - PROSE: content truncated to 80% of length (summary-like)
    - STRUCTURED: key-value format: "type: {node_type}, content: {first 200 chars}..."
    - KEY_VALUE: "id: {id}, type: {node_type}, confidence: {confidence}"
    - REFERENCE: just "{node_type}:{id}"

    Estimate token cost for each level. Return the highest level that fits in available_tokens AND is >= min_fidelity. If even REFERENCE doesn't fit, return (REFERENCE, cost) and let the caller decide.

    **Function: `def classify_into_sections(candidate: RetrievalCandidate) -> str`**
    Map node_type to section name:
    - ENTITY -> "entity_snapshots"
    - FACT, NOTE -> "stable_facts"
    - DECISION -> "recent_decisions"
    - TASK -> "active_tasks"
    - Everything else -> "provenance_refs"

    **Function: `def pack_context(scored_candidates: list[RetrievalCandidate], config: PackingConfig = DEFAULT_PACKING_CONFIG) -> MemoryBundle`**

    3-priority greedy bin-packing per RFC-0006 §5:

    1. **Calculate token costs** for all candidates using estimate_token_cost on their content.

    2. **Priority 1: Pinned + active tasks (always include).**
       Pinned = salience == 1.0. Active tasks = node_type == TASK and lifecycle_state in (TENTATIVE, STABLE).
       For each, select_representation at highest level that fits. If doesn't fit at min_fidelity, skip (budget MUST NEVER be exceeded).

    3. **Priority 2: Multi-path objects by STR descending.**
       Candidates with path_count >= 2, sorted by compute_str() descending.
       Same representation selection logic.

    4. **Priority 3: Remaining by composite score.**
       Sort by (-composite_score, str(node.id)).
       Same representation selection logic.

    5. **Assemble MemoryBundle:**
       - Group included candidates into sections via classify_into_sections()
       - Track excluded_ids for candidates that didn't fit
       - Calculate tokens_used = token_budget - overhead - remaining
       - Set budget_remaining

    **CRITICAL:** Budget MUST NEVER be exceeded. Packing stops when no more items fit. Mid-object truncation is NOT permitted — either an item fits at some representation level, or it's excluded entirely.

    **Step 1: Create `src/prme/retrieval/pipeline.py`:**

    **Class: `RetrievalPipeline`**

    Constructor:
    ```python
    def __init__(
        self,
        graph_store: GraphStore,
        vector_index: VectorIndex,
        lexical_index: LexicalIndex,
        conn: duckdb.DuckDBPyConnection,
        scoring_weights: ScoringWeights = DEFAULT_SCORING_WEIGHTS,
        packing_config: PackingConfig = DEFAULT_PACKING_CONFIG,
    ) -> None:
    ```
    Stores all backend references and config.

    **Method: `async def retrieve(self, query: str, *, user_id: str, scope: Scope | None = None, time_from: datetime | None = None, time_to: datetime | None = None, token_budget: int | None = None, weights: ScoringWeights | None = None, min_fidelity: RepresentationLevel | None = None, retrieval_mode: RetrievalMode = RetrievalMode.DEFAULT) -> RetrievalResponse`**

    This is the unified entry point per user decision. Implementation runs all 6 stages in sequence:

    1. **Stage 1 — Query Analysis:**
       `analysis = await analyze_query(query, time_from=time_from, time_to=time_to, retrieval_mode=retrieval_mode)`

    2. **Stages 2-3 — Candidate Generation + Merging:**
       `candidates, candidate_counts = await generate_candidates(analysis, graph_store=self._graph_store, vector_index=self._vector_index, lexical_index=self._lexical_index, user_id=user_id, scope=scope, config=effective_packing_config)`
       Track `embedding_mismatch` flag from the candidates module.

    3. **Stage 4 — Epistemic Filtering:**
       `filtered, excluded = filter_epistemic(candidates, analysis.retrieval_mode)`

    4. **Stage 5 — Scoring + Ranking:**
       `scored, traces = score_and_rank(filtered, effective_weights)`

    5. **Stage 6 — Context Packing:**
       `bundle = pack_context(scored, config=effective_packing_config)`

    6. **Retrieval Logging:**
       Log a RETRIEVAL_REQUEST record to the operations table:
       ```python
       import json, uuid
       op_id = str(uuid.uuid4())
       payload = json.dumps({
           "request_id": str(analysis.request_id),
           "query": query,
           "user_id": user_id,
           "candidates_generated": candidate_counts,
           "candidates_filtered": len(excluded),
           "candidates_included": bundle.included_count,
           "tokens_used": bundle.tokens_used,
           "scoring_config_version": effective_weights.version_id,
           "backends_used": list(candidate_counts.keys()),
           "embedding_mismatch": embedding_mismatch,
       })
       self._conn.execute(
           "INSERT INTO operations (id, op_type, target_id, payload, actor_id, created_at) VALUES (?, ?, ?, ?, ?, now())",
           [op_id, "RETRIEVAL_REQUEST", str(analysis.request_id), payload, user_id]
       )
       ```

    7. **Assemble and return RetrievalResponse:**
       Construct RetrievalMetadata with request_id, candidate_counts, scoring_config_version, timing, backends_used, embedding_mismatch.
       Return RetrievalResponse(bundle=bundle, results=scored, metadata=metadata, score_traces=traces).

    **Timing:** Capture start_time before Stage 1, end_time after Stage 6, compute timing_ms.

    **Weight resolution:** `effective_weights = weights if weights is not None else self._scoring_weights`
    **Packing resolution:** If token_budget provided, create modified PackingConfig. If min_fidelity provided, set on config.
