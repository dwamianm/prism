## Phase Boundary

Persist Event.scope to DuckDB and thread scope through the ingestion pipeline so memory operations can be scoped to PERSONAL, PROJECT, or ORG — not just user_id. Does not add new scope-based access control or authorization — only data persistence and propagation.
