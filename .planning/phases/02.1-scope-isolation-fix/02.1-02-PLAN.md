<objective>
Thread scope parameter through the full ingestion pipeline: extraction schema, LLM prompt, entity merger, pipeline orchestrator, and engine entry points. The LLM classifies scope per extracted object, with the ingestion-level scope as a fallback default.

Purpose: IngestionPipeline.ingest() currently has no scope parameter, so all extracted MemoryNodes get hardcoded Scope.PERSONAL. This plan makes scope a first-class parameter throughout the ingestion path, enabling scoped memory operations.

Output: Updated extraction schema with scope field, updated extraction prompt with scope classification instructions, scope parameter threaded through EntityMerger, IngestionPipeline, and MemoryEngine.
</objective>
