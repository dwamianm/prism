  <action>
Create all domain models, enums, and configuration. Follow the research patterns exactly.

**src/prme/types.py:**
- `NodeType(str, Enum)`: ENTITY, EVENT, FACT, DECISION, PREFERENCE, TASK, SUMMARY, NOTE (8 types per user decision: 7 spec + Note catch-all)
- `EdgeType(str, Enum)`: RELATES_TO, SUPERSEDES, DERIVED_FROM, MENTIONS, PART_OF, CAUSED_BY, SUPPORTS, CONTRADICTS (cover relationship types the spec describes)
- `Scope(str, Enum)`: PERSONAL, PROJECT, ORG (all three from day one per user decision)
- `LifecycleState(str, Enum)`: TENTATIVE, STABLE, SUPERSEDED, ARCHIVED
- `ALLOWED_TRANSITIONS: dict[LifecycleState, set[LifecycleState]]`: Tentative->{Stable, Superseded, Archived}, Stable->{Superseded, Archived}, Superseded->{Archived}, Archived->set() (terminal)
- `validate_transition(current: LifecycleState, target: LifecycleState) -> bool`: Returns True if transition is valid

**src/prme/models/base.py:**
- `MemoryObject(BaseModel)`: Base for all memory objects. Fields: id (UUID, default_factory=uuid4), user_id (str), session_id (str | None = None), scope (Scope, default=PERSONAL), created_at (datetime, default_factory=utcnow), updated_at (datetime, default_factory=utcnow). Use `model_config = ConfigDict(frozen=False)`.

**src/prme/models/events.py:**
- `Event(MemoryObject)`: Immutable event. Additional fields: timestamp (datetime, default_factory=utcnow), role (str -- "user", "assistant", "system"), content (str), content_hash (str -- computed from content via SHA-256 in a model_validator), metadata (dict | None = None). Override `model_config = ConfigDict(frozen=True)` since events are immutable.

**src/prme/models/nodes.py:**
- `MemoryNode(MemoryObject)`: Graph node. Additional fields: node_type (NodeType), content (str), metadata (dict | None = None), confidence (float = 0.5, ge=0.0, le=1.0), salience (float = 0.5, ge=0.0, le=1.0), lifecycle_state (LifecycleState = TENTATIVE), valid_from (datetime = utcnow), valid_to (datetime | None = None), superseded_by (UUID | None = None), evidence_refs (list[UUID] = []) -- list of event IDs providing evidence.

**src/prme/models/edges.py:**
- `MemoryEdge(BaseModel)`: Graph edge (not a MemoryObject -- edges have their own simpler ID scheme). Fields: id (UUID, default_factory=uuid4), source_id (UUID), target_id (UUID), edge_type (EdgeType), user_id (str), confidence (float = 0.5, ge=0.0, le=1.0), valid_from (datetime = utcnow), valid_to (datetime | None = None), provenance_event_id (UUID | None = None), metadata (dict | None = None), created_at (datetime = utcnow).

**src/prme/config.py:**
- `EmbeddingConfig(BaseSettings)`: provider (str = "fastembed"), model_name (str = "BAAI/bge-small-en-v1.5"), dimension (int = 384)
- `PRMEConfig(BaseSettings)`: db_path (str = "./memory.duckdb"), vector_path (str = "./vectors.usearch"), lexical_path (str = "./lexical_index"), embedding (EmbeddingConfig = EmbeddingConfig()). Set model_config with env_prefix="PRME_", env_nested_delimiter="__".

**src/prme/models/__init__.py:**
- Re-export: Event, MemoryNode, MemoryEdge, MemoryObject

All UUID fields should use `from uuid import UUID, uuid4`. All datetime fields should use `from datetime import datetime, timezone` with `datetime.now(timezone.utc)` as the factory.
