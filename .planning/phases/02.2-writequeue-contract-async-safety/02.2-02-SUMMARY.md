## Decisions Made
- Async threading pushed into provider layer (FastEmbed uses asyncio.to_thread internally) rather than being the caller's responsibility -- cleaner separation of concerns
- CachedEmbeddingProvider uses SHA-256 text hash as cache key for deduplication
- LRU eviction via OrderedDict.popitem(last=False) -- simple, zero external deps
