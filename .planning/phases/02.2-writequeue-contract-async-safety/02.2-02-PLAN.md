  <name>Task 1: Convert EmbeddingProvider to async and add CachedEmbeddingProvider</name>
  <files>src/prme/storage/embedding.py</files>
  <action>
Modify `src/prme/storage/embedding.py` to make the entire embedding layer async:

**1. EmbeddingProvider Protocol -- make embed() async:**
```python
@runtime_checkable
class EmbeddingProvider(Protocol):
    @property
    def model_name(self) -> str: ...
    @property
    def model_version(self) -> str: ...
    @property
    def dimension(self) -> int: ...

    async def embed(self, texts: list[str]) -> list[list[float]]: ...
```

**2. FastEmbedProvider -- wrap sync ONNX in asyncio.to_thread():**
- Rename current `embed()` to `_embed_sync()` (private, synchronous)
- New `async def embed()` calls `await asyncio.to_thread(self._embed_sync, texts)`
- `_embed_sync` keeps the exact current logic (_ensure_model, model.embed, tolist)

**3. OpenAIEmbeddingProvider -- native async:**
- Remove the sync `embed()` method entirely
- Rename `_embed_async()` to `embed()` (now the public method, directly async)
- Remove `asyncio.run()` call -- the method IS async now
- Keep `_ensure_client()` as-is (lazy init pattern)
- The method body: `client = self._ensure_client(); response = await client.embeddings.create(...); return [item.embedding for item in response.data]`

**4. CachedEmbeddingProvider -- new class:**
Add a `CachedEmbeddingProvider` class that wraps any `EmbeddingProvider`:
- `__init__(self, provider: EmbeddingProvider, maxsize: int = 512)`
- Properties `model_name`, `model_version`, `dimension` delegate to wrapped provider
- `_cache: OrderedDict[str, list[float]]` keyed by SHA-256 hash of text content
- `_maxsize: int` for LRU eviction bound
- `async def embed(self, texts: list[str]) -> list[list[float]]`:
  - For each text, compute `hashlib.sha256(text.encode()).hexdigest()` as cache key
  - Separate into cached hits and uncached misses
  - If uncached_texts is non-empty, `await self._provider.embed(uncached_texts)` for batch
  - Fill results from cache hits and new embeddings
  - Store new embeddings in cache, evict oldest if over maxsize (OrderedDict.popitem(last=False))
  - Return full results list preserving input order

Import additions: `import hashlib`, `from collections import OrderedDict`

**5. Update create_embedding_provider factory:**
- Wrap the returned provider in `CachedEmbeddingProvider`:
```python
def create_embedding_provider(config: EmbeddingConfig) -> EmbeddingProvider:
    if config.provider == "fastembed":
        provider = FastEmbedProvider(...)
    elif config.provider == "openai":
        provider = OpenAIEmbeddingProvider(...)
    else:
        raise ValueError(...)
    return CachedEmbeddingProvider(provider)
```

NOTE: `create_embedding_provider` return type stays `EmbeddingProvider` (CachedEmbeddingProvider satisfies the Protocol).
