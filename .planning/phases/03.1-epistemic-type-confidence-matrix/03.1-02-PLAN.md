**src/prme/epistemic/inference.py:**

   - SUMMARY nodes -> INFERRED (system-generated)
   - EVENT nodes -> OBSERVED (direct user input)
   - All other nodes (ENTITY, FACT, DECISION, PREFERENCE, TASK) -> ASSERTED (extracted from text)

2. Create `infer_source_type(node_type: NodeType, role: str | None = None) -> SourceType`:
   - If role == "user" or role == "human" -> USER_STATED
   - If role == "assistant" or role == "system" -> SYSTEM_INFERRED
   - EVENT nodes without role -> USER_STATED (default, since events are user input)
   - All others without role -> USER_STATED (conservative default)
   This is for the `store()` path. The ingestion pipeline determines source_type from conversation role.
